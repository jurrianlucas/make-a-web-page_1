Stage Zero - The Basics

The World Wide Web

The world wide is made of a lot of computers communicating with eachother. If you visit a webpage what actually happens is that your computer sends an HTTP request to a server. The server finds the HTML document and sends it to your computer where the browsers if everything goes well displays the website on your computer.

HTML Coding Basics

HTML

HTML stands for Hyptertext Markup Language. The majority of the content of the web are HTML documents. The HTML documents contain first of all text content. This describes "what you see". Secondly, HTML documents contain so-called mark up. This describes "how it looks". Additional languages and techniques can be used for advanced applications. The basis however is HTML. On Wikipedia you can find more information about HTML.

Tags and Elements

HTML is made of elements or lines of code. We tell the computer what type of element it is by using tags. A tag is distinguished from a normal text by using the opening tag and the closing tag. Should you accidentaly forget to use the tags, the browser will not recognize it as such and the result on your screen is not what you expected. On the Udacity website they tell you much more about it.

Examples of tags are for example the b-tag for bold and the em-tag for italics.

Attributes

Attributes can be used inside of an element to give it more instructions and or information about it. Attributes will always start with a name and followed by a value. An example is the so-called anchor tag that is used to create a hyper link on a web page.

Inline and Block Elements

Inline elements will only take the space that it needs to contain its content. For example making this sentence purple. Block elements create an invisbe box around around the context. The size of the block can be adjusted in width and height.

The take away

Computers are stupid! You need to tell the computer exactly what to do. I think they call this writing the correct syntax.

Stage Two - CSS

What is CSS?

CSS is code written to control the "style" of HTML elements. CSS stands for Cascading Style Sheets. An important element is inheritance. This prevents certain properties from being declared over and over again in a style sheet allowing to write less code. It enhances faster loading of webpages.
